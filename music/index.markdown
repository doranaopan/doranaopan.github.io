# どんな音楽をやってるの？

### フォルクローレ

ペルーやボリビアなどのアンデス地方で演奏される民族音楽のことを指します。
素朴な味わいのある楽曲が多いのが特徴です。日本では「コンドルは飛んでいく」などが有名です。
### ギター

弦が6本ついた楽器です。フォルクローレでは主に低音パートを担当することが多いです。
